- **Feature: `jac ai --tui` native terminal UI**: Adds an interactive terminal UI for the Jac coding agent via a jac-super plugin that loads the NA renderer (`libtui.so`) in-process by default (subprocess `jac-na-tui` fallback via `JAC_AI_TUI_BACKEND=subprocess`). Requires jac-super installed and a prebuilt or locally compiled `ai_tui_na` artifact on Linux/WSL. (jaseci-labs/jaseci#6355)
