- **Feature: `jac ai --tui` native terminal UI**: Adds an interactive terminal UI for the Jac coding agent via a jac-super plugin that loads the NA renderer in-process by default (`libtui.so` / `libtui.dylib` / `tui.dll` via ctypes; subprocess `jac-na-tui` fallback via `JAC_AI_TUI_BACKEND=subprocess` on Linux/macOS). Requires jac-super installed and a prebuilt or locally compiled `ai_tui_na` artifact. (jaseci-labs/jaseci#6355)
