- **byLLM: Ollama-style local model daemon**: `local:<alias>` models are now served by a `jac model serve` daemon that supervises bundled `llama-server` runners behind an OpenAI-compatible localhost API, with a manifest-backed model registry, CPU + Vulkan runners bundled in the binary (GPU out of the box), residency with idle eviction, and new `jac model serve` / `run` / `ps` / `show` subcommands. Weights pull from HuggingFace or a direct `url`+`sha256` (dependency-free), including Ollama-registry blobs.
