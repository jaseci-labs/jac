- **Fix: exponential slowdown in comment injection pass on deeply nested DocIR DAG**: the DocIR is a DAG where `IfBreak` break/flat branches share child objects, but `_process`, `_remove_redundant_lines`, `_get_tokens`, and `_doc_line_span` walked both sides without memoization, re-traversing shared subgraphs up to 2^k times on nested `IfBreak` nodes. Object-identity caches (`_process_cache`, `_redundant_lines_cache`, `_span_cache`) are now cleared per transform and checked at entry to each recursive method, so each node is visited at most once per pass. Benchmarked on whole-repo `jac fmt` (1306 files): **~10–20% faster** (1m47s–1m51s with fix vs 1m55s–2m18s without). Single-file overhead is negligible since the caches are only populated for shared subgraphs.
