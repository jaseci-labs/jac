- **`jac bundle` produces pure-Python wheels**: `jac bundle` now transpiles every `.jac` source to `.py` at build time and ships a wheel (and sdist) containing only Python, no `.jac` inside, so the artifact installs and imports with plain `pip` without jaclang present at runtime for code that transpiles cleanly. The `wheel` target is pure-Python by default (the interim `--target python` is removed), and a `.jac` that fails to compile aborts the build with a clear error instead of writing a partial artifact. Flat-layout projects (the default `jac create` cli scaffold, with `main.jac` at the project root and no package directory) now package correctly: their root-level sources are collected under a package named after the project with a synthesized `__init__.py`, rather than producing a metadata-only wheel.
