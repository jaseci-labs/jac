- **Ownership Phase 3: regions, sendability, and `val`/`linear` markers**: Three region- and type-level guarantees layered on the affine borrow checker, all diagnostics-only and gated so files that use none of the new surface pay nothing. A `region { ... }` block introduces a lexical arena; a reference to a value created inside a region may not escape it by being returned, stored into an outer binding, or crossed over a `flow`/`wait`, and such an escape raises `E1307`. On the native backend a region lowers to arena allocation that reuses the existing loop-iteration cleanup machinery. Borrowed (and aliased `linear`) payloads may not cross a `flow`/`wait` concurrency boundary, since a concurrent task could outlive the borrow; such a crossing raises `E1308`, while moved-in `own` values and `val` snapshots stay sendable. Two new binding markers are recognized as contextual keywords (no new reserved tokens): `val` marks a deep-immutable binding, so reassigning it, writing its fields, or taking a `&mut` through it raises `E1309`; and `linear` marks a must-use binding, so a `linear` value that is never move-consumed raises `E1305` -- the diagnostic previously reserved by the affine-`own` work is now emitted for exactly this marker.
- **Ownership Phase 3 hardening: exactly-once `linear` and container-literal region escapes**: Two soundness gaps closed. A `linear` binding is now affine move-checked like `own` (folded into the move analysis), so in addition to the `E1305` "used at least once" leak check it gets the `E1301` "used at most once" check -- together enforcing use-exactly-once, where previously consuming a `linear` value twice passed silently. Region-escape detection now also sees through container literals: a region-local returned or stored inside a list, set, tuple, or dict literal (`return [x];`, `outer = (x,);`) previously escaped undetected because only borrow and attribute-trailer chains were unwrapped; the arena checker now walks container literals (conservatively, never through function calls) and raises `E1307` for every region-local that leaks.
