- **Fix: skip `littleX` example in `test-core-runtime` CI job**: `jac/examples/littleX/littleX.test.jac` imports `numpy` and `sklearn`, but the `test-core-runtime` job only installs `jac`, `pytest`, `pytest-xdist`, `watchdog`. The example test could never actually run there - import failed at collection time and was silently dropped by `pytest_plugin.JacFile.collect`. Adding `--ignore=jac/examples/littleX` to the pytest invocation makes the gap explicit; littleX's ML-dependent tests should run in a dedicated job that has the required deps installed (or be ignored everywhere they don't). Surfaced by #5694.
