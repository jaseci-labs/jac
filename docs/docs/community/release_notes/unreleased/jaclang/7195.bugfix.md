- **`jac eject` output now boots and serves like `jac start`**: Endpoints are harvested only from the entry module's import closure (standalone `with entry` scripts no longer hijack uvicorn at boot), `@restspec(path=...)` routes are mounted at their declared paths, the client runtime's sibling chunks and compiled `pages/` router are copied so the frontend renders, and the dev proxy targets `127.0.0.1` instead of `localhost` to avoid IPv6 `ETIMEDOUT` stalls.
