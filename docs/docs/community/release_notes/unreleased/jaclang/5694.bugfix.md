- **Fix: Surface Jac test-collection import errors instead of silently dropping them**: `pytest_plugin.JacFile.collect` previously caught every exception during `importlib.import_module` and returned an empty list, so a Jac file that failed to compile was reported as "no tests collected" rather than a real collection error. The exception now propagates and pytest renders it as a normal `ERROR <file>` collection report with the original traceback, making genuine compile regressions visible in CI.
