- **Faster hook dispatch via codegen and devirtualization**: The `Jac.*` runtime hook facade no longer rebuilds its arguments with `inspect.Signature.bind_partial` on every call. Phase 1 emits a real-signature proxy per hook (so CPython binds arguments natively instead of via `inspect`); Phase 2, sealed once at the first `ExecutionContext`, devirtualizes single-implementation hooks to a direct call while multi-implementation hooks stay on the dynamic dispatcher. On a dispatch-bound graph traversal this is about 2x (Phase 1) to 2.6x (Phase 1+2) with no behavior change. Both layers are toggleable for ablation via the `JAC_HOOK_CODEGEN` and `JAC_HOOK_SEAL` environment variables (default on).
