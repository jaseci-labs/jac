- **Native: `str()` on an `any`-boxed scalar now returns the value, not the type tag**: In the native backend, calling `str()` on an `any`-boxed scalar (e.g. a value read from a `dict[str, any]`) emitted the `JacVal` discriminant tag byte instead of the value. `str()` now unboxes the `JacVal` at runtime and renders str/int/float/bool/None correctly.
