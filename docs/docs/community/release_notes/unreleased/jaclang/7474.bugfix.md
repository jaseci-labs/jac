- **Fix: `sv import` inside a `cl { }` block broke the client build**: a `sv import` should compile to a network call to the server. Nested inside a `cl { }` block it instead compiled to a plain JavaScript import of the server module -- a file that is never built for the client -- so the bundle failed with an unresolvable import from Vite, even though `jac check` passed. The block was overwriting the context of every statement inside it, including one that had already set its own with an explicit `sv` prefix. An explicit prefix now wins over the enclosing block.
