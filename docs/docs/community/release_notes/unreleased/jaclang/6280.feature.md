- **Feature: `jac build` whole-program builder + `.jab` artifact**: `JacCompiler.build()` is now a real whole-program builder that walks the transitive server import closure (BFS over `uni.Import`), runs the full type-check schedule across every reachable user module, and gates on any user-module error (returns `None` instead of emitting a partial artifact). A new `jac build` command compiles a project into a single relocatable `.jab` (Jac App Bundle): a `FLAG_PROGRAM` JIR container whose `SEC_MODULE_TABLE` inner-frames each module's bytecode/MTIR/interop, alongside a `SEC_MANIFEST` and `SEC_DEPGRAPH`. Supports `--profile`, `--out`/`-o`, `--target`, `--no-typecheck`, and `--check-only`.
