- **Fix: `--experimental` host build finds the source checkout reliably**: `host_repo_root()` no longer derives the monorepo root from a fixed `__file__` depth, which broke after the scale fold relocated the injector and which mis-resolved entirely when jaclang runs from the bundled jac runtime cache (the binary-ship path shipped `zig build` a nonexistent `<cache>/jac` cwd). It now prefers a `JAC_REPO_ROOT` override, then searches upward from `cwd`/`__file__` for the `jac/build.zig` marker, fixing the no-Docker `--experimental` deploy and its k8s e2e.
