- **Feature: Expose pip install flags via `jac install`**: Adds `--upgrade` (`-U`), `--force-reinstall`, `--no-cache-dir`, `--pre`, `--dry-run`, `--no-deps`, `--quiet` (`-q`), and `--prefer-binary` flags to `jac install`, passing them through to the underlying pip invocation. The `jac update` command continues to apply `--upgrade` automatically. Internally, `DependencyInstaller` now accepts an `install_flags: list[str]` field in place of individual boolean fields, and CLI argument names with underscores are now correctly registered as kebab-case flags (e.g. `--no-cache-dir` instead of `--no_cache_dir`).
