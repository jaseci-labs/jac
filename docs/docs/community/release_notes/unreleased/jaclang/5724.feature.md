- **`root` is a SpecialVarRef again**: `root` is restored as a reserved keyword (`KW_ROOT`) and parses as a `SpecialVarRef`, mirroring how `here` and `visitor` are bound. The type checker resolves it directly to `Root`, the binder forbids local rebinding, and codegen lowers it to `Jac.root()`. Bare `root` is the canonical form; `root()` no longer carries deprecation weight (W0062 is removed) and the `def root() -> Root` ambient declaration in `jac_builtins.pyi` is dropped. Backtick escaping (`` `root ``) is required to use `root` as a regular identifier.
