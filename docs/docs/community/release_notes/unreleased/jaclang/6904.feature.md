- **Build: bundled CPython 3.14 and leaner typeshed**: the single `jac` binary now embeds Python 3.14 (python-build-standalone `cpython-3.14.6`, `llvmlite>=0.47`) and ships only the typeshed **stdlib** stubs. Third-party type stubs are no longer bundled: install the matching `types-*` package into your project (`jac add types-foo`) and the checker resolves it via PEP 561 `<pkg>-stubs` from the `.jac/venv`, so types track the installed version instead of a vendored snapshot. The typeshed git submodule is removed; stdlib stubs are fetched at a pinned commit (the `fetch-typeshed` step of `zig build`), gitignored rather than committed. Running from a source checkout now needs a one-time `zig build fetch-typeshed`, which a full `zig build` also runs automatically.
