- **Scale: fat bundles ship the python dep closure inside the .jab**: pack_jab vendors the app's pypi wheel closure into `_vendor/` (resolved by the same binary pods run, against the pod's sanitized jac.toml) and pod bootstrap installs it offline with `PIP_NO_INDEX`, so pods never touch PyPI at boot; on by default with a graceful thin-bundle fallback, `[scale] fat_bundle = true` makes vendoring failures fatal, `false` opts out.
