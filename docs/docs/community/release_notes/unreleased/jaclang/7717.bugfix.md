- **Fix: client builds no longer require node**: `bun x vite` resolves to `node_modules/.bin/vite`, a shim carrying Vite's `#!/usr/bin/env node` shebang, so bun handed the process off to node and every client build and dev server failed on a machine without node installed. Since jac provisions bun and never provisions node, this broke an otherwise valid setup. Vite's local JS entry is now resolved and invoked as `bun <entry>`, keeping the run inside bun, with the previous `bun x vite` behavior kept as a fallback when no local install is present.
- **Fix: strip-comments no longer deletes `jac:ignore` directives**: The W3050 `strip-comments` auto-lint rule treated every comment as slop, including `# jac:ignore[...]` lines that the parser reads into inline suppressions. Stripping one silently re-armed the diagnostic it was suppressing. Comments on suppression lines are now preserved.
