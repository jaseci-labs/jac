- **Feature: Rust-to-Jac FFI bridge**: `import from rust.<crate>` resolves Rust crates at both nacompile time and CPython runtime. A proc-macro (`#[jac_export]`) annotates Rust functions for export; a binder auto-generates the Jac wrapper module from the crate's type graph, including owning smart-pointer wrappers (via `ouroboros`) for self-referential types and a callback vertical for `replace_all`-style closures. The na codegen lowers callback trampolines with a `{call, ctx}` closure ABI so Jac callbacks can be passed directly to Rust.
