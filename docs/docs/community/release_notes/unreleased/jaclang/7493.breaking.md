- **Breaking: byLLM local models replace in-process llama-cpp with a bundled runner**: `local:<alias>` runs via the `jac model serve` daemon instead of in-process `llama-cpp-python`. `jac install 'byllm[local]'` no longer exists (the runner ships in the binary). Un-pulled models error with `Run: jac model pull <alias>` instead of auto-downloading. `[byllm.local].default_alias` moved to `[byllm.serve].default_alias` (old key honored as a fallback); `[byllm.local]` is now `llama-server` launch flags.
