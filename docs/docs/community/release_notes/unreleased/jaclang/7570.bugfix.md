- **Type Checker: no more false `E1052` on calls to `-> Callable` decorated functions**: A decorator whose resolved type was a bare/gradual `Callable` (e.g. `-> Callable`, or a `Callable[[Callable], Callable]` factory) collapsed the decorated function's signature, so every call-site argument was flagged (`E1052` on kwargs, `E1051` on positionals). Bare/gradual `Callable` decorators now preserve the original concrete signature; signature-transforming decorators and genuine argument errors are unaffected.
