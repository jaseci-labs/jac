- **Fix: `jac install -e <path>` resolves config from target package when run outside its directory**: `get_config()` previously searched from the current working directory and was subject to the module-level cache, returning the wrong config when invoked outside the target package. The install command now uses `get_config(start_path=<resolved-path>, force_discover=True)` to correctly discover the target package's config regardless of the working directory.
