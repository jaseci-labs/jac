- **Fix(jac2c): module-qualified type identity so same-named types across modules no longer collide**: `get_shared_type_facts` and `get_shared_layout_registry` merged each module's facts/layout first-wins by bare archetype name, so two modules each declaring (say) `Token` or `State` (inevitable across a large program) silently resolved to whichever module was indexed first, inheriting the other's fields and ref-ness with no diagnostic, and both emitted a colliding `struct Token`. jac2c now assigns every archetype a program-unique `tid` via a shared `TypeIdIndex`: a name that is unique across the whole module set keeps its bare name (single-module emission is byte-identical), while a collided name is mangled per module (`main__Token` / `lib__Token`). The shared TypeFacts, LayoutRegistry, C struct/constructor/method/dtor/vtable names, and runtime type ids are all keyed by tid, and every type reference (local declarations, `import from ... { T as U }` aliases, and cross-module return types) resolves through its defining module's scope, so imported `extern` declarations match the definitions they name. Proven by `test_jac2c_name_collision`: two modules each declaring `obj Token` with different shapes and a different `score()` emit distinct structs and link into a binary whose output matches the stock interpreter.
- **Fix(jac2c): typed exception-object payloads**: `raise MyErr(...)` for a user exception archetype now constructs the object and throws it as the RC-owned payload (previously it threw only the bare first constructor argument), and `except MyErr as e` binds `e` as a real `struct MyErr *` (previously an untyped `void *`, which made `e.field` invalid C) whose fields the handler can read. The taken payload is released exactly once on every handler exit path (return and fallthrough), including when the raise crosses a call boundary into an outer handler, so the live-object count stays balanced. Builtin exceptions keep their existing first-argument payload behavior. Verified by `test_jac2c_runtime` against the stock interpreter with a trailing `jac_rc_live() == 0` balance check.
