- **Refactor: FieldPredicate as typed backend-neutral predicate protocol**: The compiler now emits typed `FieldPredicate(op, value)` objects (with plain operator names: `lt`, `lte`, `gt`, `gte`, `eq`, `ne`, `in`, `nin`) instead of MongoDB-flavored dicts for `[?:Type, field op const]` filters. `QueryPlan.field_predicates` is typed as `dict[str, list[FieldPredicate]]`. Backends translate to their native syntax at query time; `FieldPredicate` lives in `jac0core.filter_spec` to avoid circular imports.
