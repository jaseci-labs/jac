- **Feature: `--experimental` downloads the prebuilt dev binary when available**: deploying `jaseci-labs/jaseci@main|dev` with `--experimental` now downloads and sha256-verifies the matching per-platform `jac-dev-linux-<arch>` release asset for the pod's architecture instead of host-building it from source, falling back to the host-build for forks, feature branches, pinned commits, or an undetectable arch. Pod arch is read from the cluster nodes, which also sidesteps cross-arch builds.
