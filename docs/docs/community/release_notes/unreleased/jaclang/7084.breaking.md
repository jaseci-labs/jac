- **Breaking: `jac start --scale --experimental` is removed**: pod runtime selection moved from the CLI flag to the `[dev]` jac.toml stanza, and the deploy no longer builds jaclang at all (it downloads a prebuilt binary). To run your own jaclang edits in-cluster, add `[dev] jaclang_source = "<dir containing jaclang/>"` (or `[dev] repo`/`branch`) - the source is overlaid on the downloaded dev binary via the editable dev loop. No `[dev]` runs the latest stable release as-is.
