- **Fix: `T | None` narrowing now propagates into comprehensions assigned inside `if`-blocks**: `_wire_compr_filters` in `CFGBuildPass` no longer skips `Assignment` nodes when searching for the enclosing CFG node. This makes `IfStmt -> Assignment -> InnerCompr` the CFG chain instead of the incorrect `IfStmt -> InnerCompr` direct link, so `_classify_edge` correctly identifies the true branch and flow narrowing reaches the comprehension's iterable. Eliminates false-positive `E1030` errors on attribute access inside comprehensions like `result = [f.id for f in p.following]` inside an `if p` block.
