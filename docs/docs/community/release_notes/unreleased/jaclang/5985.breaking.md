- **`defview` is gone; use `def:pub Name(...) -> JsxElement { return {...}; }` with JSX statement slots**: the standalone `defview` declarator has been removed in favor of regular function declarations whose JSX body uses the new `{...}` statement-slot form. Migration is mechanical -- the slot does the work the `defview` body used to do, just at a child position. Migrate `defview Name { ... }` -> `def:pub Name() -> JsxElement { return <>{...}</>; }` where each top-level JSX statement inside the original body becomes a statement-slot child.
