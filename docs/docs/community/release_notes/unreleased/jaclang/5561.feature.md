- **Persistence: Schema Drift Just Works**: Editing a `node`/`obj`'s fields no longer forces wiping `.jac/data/`. `SqliteMemory` now stamps each row with a per-archetype fingerprint, tolerates added/removed fields, coerces primitive type changes, and quarantines truly unloadable rows. Class renames are handled by the `@archetype_alias` decorator. Legacy pickle DBs auto-migrate on first open.
- **CLI: `jac db` Operator Toolkit**: New `jac db` command group: `inspect`, `quarantine list`/`show`, `alias list`/`add`/`remove`, and `recover`/`recover-all`. Works with both `SqliteMemory` and `MongoBackend`.
