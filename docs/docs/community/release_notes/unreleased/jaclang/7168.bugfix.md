- **Bugfix: Rust bridge cold-cache import**: `jaclang.compiler.rust_bridge._finder` failed to parse on a fresh checkout because its version-sort `key` lambda used Python syntax instead of Jac lambda syntax. The `rust_bridge` package (imported eagerly via `__init__.jac`) was therefore unimportable until the compiler cache warmed. The lambda is now written in Jac syntax, so cold-cache import succeeds.
