- **Scale deploy: host client build runs in parallel with the app seal, using the downloaded pod binary when the driver has no jac on PATH**: The client bundle build no longer runs serially before the runtime prep. It starts as soon as the pod binary is downloaded, overlaps the app precompile/seal (they write disjoint paths), and is joined right before the bundle is packed. When the deploy driver cannot resolve its own `jac` executable (CI runners, python-driver environments), the build now falls back to the just-downloaded pod binary if the host can execute it, instead of silently deferring a multi-minute client build to the gateway pod's boot.
