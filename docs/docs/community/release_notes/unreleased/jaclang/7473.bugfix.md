- **Bugfix: the official base image (jaseci/jaclang) now bakes setuptools into its embedded runtime**: pods on the image that install a dependency without a wheel for the runtime's Python fell back to an sdist build and pip failed the entire install with `BackendUnavailable: Cannot import 'setuptools.build_meta'`, crash-looping every service. The image's seed install now bakes setuptools into the runtime site (the venv-path self-seed from #7084 never applied to runtime-site installs), and the image build asserts it landed.
