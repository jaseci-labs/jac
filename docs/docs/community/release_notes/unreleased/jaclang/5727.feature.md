**Feat: Native `jac publish` pipeline with `jac.toml`**: Replaces `pyproject.toml` with `jac.toml` as the project manifest for Jac packages. Adds `jac publish` command that validates, precompiles `.jac` → `.jir` bytecode, and builds a standards-compliant Python wheel (`pip install`-ready) entirely in Jac - no `pyproject.toml` or `setuptools` required.
