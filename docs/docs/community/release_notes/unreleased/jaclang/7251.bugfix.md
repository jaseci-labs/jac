- **`jac eject` no longer ships test code or developer paths in the backend**: The compiler merges `.impl.jac` / `.test.jac` annexes into their parent module, so ejected backend modules previously carried `@jac_test` functions and `@impl_patch_filename('/abs/path/...')` decorators that leaked the author's filesystem layout. Each emitted module is now sanitized: test functions are dropped, the dev-path decorators are removed (tracebacks resolve to the emitted `.py`), and the unused `jac_test` / `impl_patch_filename` imports are pruned.
