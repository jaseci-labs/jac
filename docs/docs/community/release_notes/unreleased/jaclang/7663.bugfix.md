- **Fix: web dev server failed on first run for `web-static`/`web`/PWA projects**: `jac start -d` on a brand new project failed with `Failed to resolve import "/compiled/_entry.js"`, because `WebTarget.dev()` started the Vite dev server without ever compiling the `.jac` client entry into `.jac/client/compiled/` first; it only worked once a prior `jac start` (production build) had populated that directory as a side effect. `WebTarget.dev()` now compiles the client entry up front and watches `.jac` files for HMR recompilation, matching the pattern `MobileTarget.dev()` and `NativeDesktopTarget.dev()` already used. Since `StaticTarget` and `PWATarget` both inherit `WebTarget.dev()`, this also fixes dev mode for `web-static` and PWA projects.
