- **Fix: `jac start -d` failed on a fresh `web-static` or PWA project (#7660)**: dev mode started the Vite server without ever compiling the `.jac` client entry into `.jac/client/compiled/`, so the first run failed with `Failed to resolve import "/compiled/_entry.js"` and rendered a blank page. It only appeared to work once a prior `jac start` production build had populated that directory as a side effect. `WebTarget.dev()` now compiles the client entry up front and watches `.jac` files for HMR recompilation, matching the pattern `MobileTarget.dev()` and `NativeDesktopTarget.dev()` already used; `StaticTarget` and `PWATarget` inherit the fixed flow.
