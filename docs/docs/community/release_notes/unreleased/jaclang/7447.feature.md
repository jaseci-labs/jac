- **Perf: Fast-path `jac --version`**: `--version` is now resolved before the full CLI is imported, dropping it from ~1.3s to ~0.5s. Project config, provider, and template initialization is deferred until after help/version exits, so ordinary commands also skip that work.
- **Perf: On-demand feature command loading**: Feature plugins (client/scale/shadcn/byllm) are now imported only when needed: all of them for bare/`--help`, just the matching one for `jac setup`/`scale`/`retheme`/`model`, and none for ordinary commands, saving ~260ms on every non-feature invocation. Routing is centralized in `register_feature_commands(raw_argv)` backed by a declarative `FEATURE_COMMAND_MODULES` map, so adding a feature is now a one-line data entry.
