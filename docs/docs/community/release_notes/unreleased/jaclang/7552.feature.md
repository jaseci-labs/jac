- **Borrow-checked headless shooter benchmark (`jac/examples/raylib_shooter/shooter_headless.na.jac`)**: the raylib cube shooter's game loop annotated to the enforced zero-RC endpoint (`own` / `&` / `&mut` borrows, target/bullet pools as parallel arrays inside one owned `World` in the ownbench index-arena idiom), compiled headerless with `--enforce-nogc --gc none --assert-no-rc` -- no collector, machine-checked zero reference counting. A deterministic scripted pilot replaces the keyboard/mouse and a virtual renderer folds the exact `draw_floor`/`draw_box` vertex stream into an integer digest, so it needs no window, GPU, or raylib download. `demo.sh --headless` builds the same unchanged source under all three memory modes (`none`/`rc`/`cycles`) and requires byte-identical digests across them -- the ownbench erasure/identity discipline applied to a real game loop (this suite is what surfaced the subscript aug-assign miscompile).
