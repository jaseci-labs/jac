- **Opt-in ownership & borrow checker, plus a native ARC cycle collector**: A new opt-in ownership layer on Jac's managed RC/GC memory, entered only via annotations: `own` (move-checked binding), `&` (shared borrow), and `&mut` (exclusive borrow). A new `OwnershipCheckPass` enforces move-after-use, borrow exclusivity (many shared or one mutable, never both), and borrow escape (a borrow may not be stored in a field or outlive its scope) with diagnostics `E1301`-`E1304` and `E1306`, on every backend. Ownership is affine (an unconsumed `own` binding is dropped, not an error); only `obj` and value/primitive types are ownable, while `node`/`edge`/`walker` stay permanently managed. The native backend adds a synchronous Bacon-Rajan cycle collector that reclaims reference cycles (on by default, auto-sweeping at 1000 purple roots; `JAC_NO_GC` disables all RC) and elides the retain/release pair for proven-unique moves.
