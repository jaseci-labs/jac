- **Client Compiler: `@jac/*` modules resolve via entry points**: Packages can register virtual `@jac/*` modules through the `jac.modules` entry-point group so `@jac/desktop` and similar SDKs resolve without vendoring into jaclang. The Vite bundler now compiles a resolvable `@jac/desktop` SDK to `compiled/desktop_api.js` and wires the alias when present.
- **Native Compiler: SubTag type annotations lower correctly**: Native `_resolve_jac_type` unwraps `SubTag` wrappers so annotated globals and clib fields (e.g. `Callable[...]`) resolve to the correct LLVM types without re-reading `type_tag.tag`.
