- **`py2jac` roundtrips class keyword arguments**: class-definition keyword arguments (e.g. `metaclass=Meta`, PEP 487 `__init_subclass__` kwargs) now survive the Python -> Jac -> Python pipeline instead of being dropped. The `Archetype` node carries a `keywords` field, the parser splits `KWPair` keywords from positional base classes in archetype base lists, and pyast load/gen plus the normalize/unparse tool passes thread them through end to end.
