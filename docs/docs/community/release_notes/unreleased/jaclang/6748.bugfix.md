- **Fix: `jac install <pkg>` now uses uv when available**: Direct package installs (`jac install numpy`) were hardcoding bare `pip` and bypassing the `pip_command()` helper, so `uv` was never used even when it was on `PATH`. This also caused `--no-uv` and `JAC_NO_UV=1` to be silently ignored for this code path. All install paths now consistently prefer `uv pip` when available.
