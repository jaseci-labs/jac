- **Feature: GraphMend graph-break detection (#6740)**: Adds the opt-in `--graphmend` compiler flag and a graph-break detection pass. This is the first slice of GraphMend, a technique that eliminates PyTorch 2 FX graph breaks. The pass locates `torch.compile` entry points and tags fixable break candidates (data-dependent control flow, validation guards, and `print`/logger side effects) via use-def analysis over the symbol table. Detection only and inert unless `--graphmend` is set; the transformation passes follow in later PRs.
