- **Scale: gateway pods stop installing the npm closure when the client ships prebuilt**: npm dependencies exist only to build the client bundle; when the deploy ships a host-built `.jac/client/dist`, the gateway serves it and never builds, so its bootstrap now runs `jac install --no-npm` (~10s off every gateway boot, restart, and replica scale-up). Service pods switch from the jac.toml table-strip to the same flag, which also skips the client machinery's base npm closure that the strip missed on client-kind projects. Bundles without a prebuilt dist keep the full install so the on-boot build fallback still works.
