- **Scale: gateway pods stop installing 500+ npm packages when the client ships prebuilt**: npm dependencies exist only to build the client bundle; when the deploy ships a host-built `.jac/client/dist`, the gateway serves it and never builds, so its bootstrap now strips the npm closure exactly like service pods do (~10s off every gateway boot, restart, and replica scale-up). Bundles without a prebuilt dist keep the install so the on-boot build fallback still works.
