- **Lint: W3039 Auto-Convert `.map(lambda → JSX)` to List Comprehension**: `jac lint --fix` now rewrites verbose React-style `.map()` lambda patterns into Jac's native JSX comprehension syntax. `items.map(lambda item: any -> any { return <li>{item}</li>; })` becomes `[<li>{item}</li> for item in items]`. When the lambda includes an `int`-typed index parameter, `enumerate()` is injected automatically: `[<li key={idx}>{item}</li> for (idx, item) in enumerate(items)]`. The rule is suppressible via `# noqa: W3039` or `map-lambda-to-comprehension` in `jac.toml`.
