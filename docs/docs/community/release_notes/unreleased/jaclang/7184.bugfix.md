- **Fix: slice assignment rejected valid buffers**: Subscript assignment (`container[index] = value`) now type-checks by dispatching `__setitem__` on the index type instead of comparing the RHS against the `__getitem__` read type, so valid slice writes whose RHS is broader than the container — e.g. `bytearray[0:4] = b"\x7fELF"` or a `list[int]` slice assigned an iterable of ints — no longer raise a spurious `E1001`.
