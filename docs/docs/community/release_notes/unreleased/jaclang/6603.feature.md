- You can now finish a schema migration instead of carrying `__jac_schema__` drift rules forever: `jac db schema census` shows how many stored documents each rule still applies to, `jac db migrate` rewrites every document into the current shape (with `--dry-run` to preview), and once a rule's count reaches zero `jac db schema retire <Class>` deletes the now-dead hook from your source (preview by default, `--write` to apply, and it refuses unless no document still needs the rule).
