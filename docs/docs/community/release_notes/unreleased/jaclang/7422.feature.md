- **Feature: Dev devtools LLM-call capture**: When the devtools overlay is active (dev only, gated by `JAC_DEVTOOLS`), the base HTTP server now records the byllm LLM calls made during each walker/function request and attaches them to that request's response as `llm_calls`, so the overlay can show, per request, the model, latency, prompt, and response of each call. It is inert when byllm is not installed and is omitted from production.
