- **Scale deploy: sealed .jab cache**: the driver caches the sealed app bundle by a digest of its inputs (source content, database intent, pod binary), so redeploys of unchanged content skip the app seal entirely - measured on EKS as the host phase dropping from ~150s to ~76s, with the byte-identical bundle also deduping the PVC seed. Any source or binary change re-keys and re-seals as before.
