- **Official jaclang Docker images**: every release now publishes `ghcr.io/jaseci-labs/jaclang:<version>` and `:latest`, and every main push refreshes the rolling `:dev` tag - multi-arch (amd64/arm64), built from the same binaries as the release assets. The image ships with jac's runtime already extracted, so containers skip the one-time setup and the first command starts instantly, for any user id. `docker run --rm -v "$PWD":/app ghcr.io/jaseci-labs/jaclang:latest run main.jac` just works.
