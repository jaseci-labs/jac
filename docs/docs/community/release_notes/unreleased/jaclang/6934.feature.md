- **Native: partial-native classes / per-method opt-out**: A native-eligible class may now contain methods the native backend cannot lower (e.g. one using `eval`) without poisoning its whole module's native compile. Such a method is demoted to its Python implementation while the class keeps its native layout, native constructor, and lowerable sibling methods; construction-critical `init`/`postinit` gaps remain hard errors. Set `JAC_NATIVE_DEBUG=1` to see which methods were demoted.
