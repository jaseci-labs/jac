- **CLI: static command dispatch + native launcher fast-path**: `jac` now routes commands through a generated static dispatch table (`command_dispatch.jac`, kept fresh by the `jac gen-cli-dispatch --verify` CI step) instead of resolving each invocation through the runtime command registry/executor, and the native launcher fast-paths hot commands directly to their handler. This removes dynamic indirection from the CLI hot path with no change to any command's flags or behavior.
