- **App precompile no longer inherits the driver's dev overlay**: `_precompile_app`'s subprocess ran in the driver's CWD, so a `[dev] jaclang_source` in the driver's `jac.toml` made the pod binary recompile the whole jaclang runtime before compiling the app (~4.5 min wasted per deploy; 72s standalone vs 5m26s in-deploy measured). The subprocess now runs from the staging dir with `JAC_NO_DEV_SOURCE=1` and a scrubbed python env - also the correct semantics, since pod-keyed `.jir` must come from the pod binary's own runtime.
