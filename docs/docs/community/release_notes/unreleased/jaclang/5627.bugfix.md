- **Fix: Suppress spurious W1051 for intrinsic JSX tag names**: The type checker no longer emits false W1051 ("Expression type could not be resolved (Unknown)") warnings for intrinsic HTML element tag names (`div`, `h1`, `button`, etc.) in JSX expressions. These tags are resolved via `_JsxIntrinsicElements` in the JSX element handler, not through normal scope lookup, so the `exit_name` handler now skips `JsxElementName` children to avoid the false positive.
