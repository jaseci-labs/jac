- **Scale: gateway-owned identity for microservice fleets**: fleet services now accept gateway-issued session tokens by JIT-provisioning the user and root on first authenticated use (gated by `[scale.microservices.identity] gateway_owned`, default on), and the gateway's core `/user/*` endpoints are excluded from builtin passthrough by name - fixing the cross-service auth break where every authenticated walker 401'd because users existed only in the gateway's store.
