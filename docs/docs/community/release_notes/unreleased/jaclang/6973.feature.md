- **Feat(c2jac): C-to-Jac transpiler with honest tiered lowering**: Implements `jac c2jac <file.i>` - a transpiler from preprocessed C to Jac, reusing the existing `PyastBuildPass` (pycparser C-AST -> CPython `ast` -> uni IR) with a direct-IR path for constructs with no Python-AST analog (C `for`, enums). Every node is routed on two axes - **Mechanism** (`PY_AST` vs `DIRECT_IR`) x **Tier** (Tier-A faithful vs Tier-B best-effort) - and the verdict is written to a sidecar `<file>.c2jac.report.json` for full transparency. Coverage includes switch/do-while/ternary lowering, faithful for-body lowering, and §7 Tier-A pointer idioms (list-of-objects, ptr-as-ref, realloc-elide).
