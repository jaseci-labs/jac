- **Fix: `jac0` codegen for empty `match` / `switch` bodies and misordered `default:` arms**: The bootstrap transpiler used to emit invalid Python in three latent edge cases. (1) `match X {}` produced `match X:` with no cases (Python `SyntaxError`). (2) `switch X {}` produced no output at all, silently dropping the subject expression along with any side effects. (3) A `switch` whose `default:` arm appeared before any value arm (or whose only arm was `default:`) emitted `else:` with no preceding `if`/`elif`, again invalid Python. The codegen now always evaluates the subject, splits the default arm out, and places it at the end of the `if`/`elif`/`else` chain, so source-order placement of `default:` no longer breaks compilation.
