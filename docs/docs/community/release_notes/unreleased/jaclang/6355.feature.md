- **Feature: `jac ai --tui` native terminal UI**: Adds `--tui` to `jac ai` (mutually exclusive with `--ui`), `AgentRequest.tui` routing, and terminal-output gating during TUI sessions so agent prints do not corrupt the alt-screen. `--tui` ships on a single self-hosting backend: the dispatch `execve`s the bundled `jac-ai-tui` embed host (`ai_tui_na/host_embed.na.jac`, built by `build_embed.sh` on first run or pre-baked into the fused payload), which runs the byLLM agent in an embedded CPython and the linked NA renderer in one process, booting fully offline when byLLM is bundled. The earlier in-process (`libtui.so`/`libtui.dylib` via ctypes) and subprocess (`jac-na-tui` sidecar over pipes) transports, and the `JAC_AI_TUI_BACKEND` selector, are retired; `tui_host.jac` / `libtui.so` survive only as the renderer's test harness.
