- **Feature: `jac ai --tui` native terminal UI**: Adds `--tui` to `jac ai` (mutually exclusive with `--ui`), `AgentRequest.tui` routing to the bundled NA renderer, and terminal-output gating during TUI sessions so agent prints do not corrupt the alt-screen. The default backend loads `libtui.so` / `libtui.dylib` / `tui.dll` in-process via ctypes; set `JAC_AI_TUI_BACKEND=subprocess` to spawn the `jac-na-tui` sidecar over pipes (Linux/macOS). Requires a prebuilt or locally compiled `ai_tui_na` artifact (`build.sh`). (jaseci-labs/jaseci#6355)
