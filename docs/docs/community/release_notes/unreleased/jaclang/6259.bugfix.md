- **Fix: false E1030 on metaclass-provided attributes like `Enum.__members__`**: Accessing `Color.__members__` raised `E1030: Attribute '__members__' not found` even though the code is valid. These attributes live on `EnumMeta` (the metaclass), not in the class MRO. The fix captures the `metaclass=` keyword from Python stubs, propagates it to subclasses, and falls back to the metaclass MRO during attribute lookup - working for any metaclass, not just enums.
