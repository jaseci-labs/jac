- **Scale deploy: ship the prebuilt admin console to pods**: The deploy driver now stages the admin console into the app bundle at `.jac/admin`, so the gateway serves it directly instead of building it from source on every boot. On EKS this cut the gateway's main-container ready time from ~163s to ~23s, and the same ~140s is saved again on every gateway restart or scale-up. The console is built once on the host (cached in the jaclang package) and staging is best-effort: if it fails, the gateway falls back to the on-pod build.
