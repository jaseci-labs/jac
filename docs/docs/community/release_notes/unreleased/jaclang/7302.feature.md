- **Scale: the jac binary ships the admin console prebuilt**: The binary release pipeline now builds the admin console once per platform and bakes its `_dist` into the runtime payload, so a scale gateway serves it directly instead of rebuilding it from source on every pod boot. On EKS this cut the gateway's main-container ready time from ~163s to ~23s, and the same ~140s is saved again on every gateway restart or scale-up. For pod binaries released before the bake, the deploy driver stages its own package's `_dist` into the app bundle at `.jac/admin` as a fallback; if neither is available the gateway builds on boot exactly as before.
