- **Feature: MobUI target enhancements**: `@jac/mobui` now re-exports `KeyboardAvoidingView`, `Keyboard`, `Platform`, and `ActivityIndicator`; the React Native target writes and imports a `__jacApiBase.js` shim so the native entry resolves the build-time API base URL before any bundle code runs; `jacSignup` failures now include the HTTP `status` so callers can tell identity conflicts from server errors; and a new `jac-mobui` guide/skill documents the cross-platform primitives, the no-HTML rule, and the `jac start --client react-native` build. The worked product-scale reference app built on these (`jachammer`, a full mobile + web client for jacBuilder) lives in the [jacBuilder repo](https://github.com/jaseci-labs/jacBuilder) under `mobile/`.
