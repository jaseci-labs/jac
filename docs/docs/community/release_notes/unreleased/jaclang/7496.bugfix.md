- **Fix: `jac lint` no longer runs the AST-rewriting auto-lint pass**: the report path (`jac lint`, `JacProgram.jac_file_linter`) used `JacAutoLintPass` (report_only=False), performing the full auto-lint mutation pass and discarding the result. It now uses `JacLintCheckPass` (report_only=True), matching `jac check`. The lint pass is ~6x faster and `jac lint` ~19% faster end-to-end over the tracked tree, with identical warnings.
