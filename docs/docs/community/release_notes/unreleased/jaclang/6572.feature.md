- **Feature: Exported symbols for native executables**: `jac nacompile` can now export defined symbols into an executable's `.dynsym` so they resolve via `dlsym`/`RTLD_NEXT` (e.g. exporting `close()` for the CEF zygote), and appends `libc.so.6`/`libm.so.6` last in `DT_NEEDED` so libcef's `.init_array` `dlsym(RTLD_NEXT, "close")` resolves correctly.
