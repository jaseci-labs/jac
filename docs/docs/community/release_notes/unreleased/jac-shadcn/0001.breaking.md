- **Breaking: all shadcn commands moved to `jac shadcn` namespace**: `jac add --shadcn` / `jac remove --shadcn` flags removed. Use `jac shadcn add`, `jac shadcn remove`, `jac shadcn init`, etc.
- **Breaking: template alias renamed `jac-shadcn` → `shadcn`**: `jac create myapp --use jac-shadcn` → `jac create myapp --use shadcn`.
- **Breaking: `jac shadcn init` no longer accepts a URL**: use declarative flags — `jac shadcn init --style nova --theme rose --font outfit`.
- **Breaking: default install set is now 10 essentials, not ~53**: pass `--all` to restore the old "every component" behavior, or `--bare` for none.
- **Breaking: `menuColor` config field removed**: never honored by the registry. Drop it from `[jac-shadcn]` in `jac.toml`.
- **Breaking: `init_shadcn_theme(config, theme_url)` Python signature changed** to `init_shadcn_theme(config, *, style, base_color, theme, font, radius, menu_accent, install_set)`. Affects direct API callers only.
