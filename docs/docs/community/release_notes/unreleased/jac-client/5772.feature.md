- **Feat: SSE streaming primitives for walkers and `def:pub` functions**: Added `jacSpawnStream(walker, nodeId, fields, onEvent)`, `jacCallFunctionStream(name, args, onEvent)`, and the React-style `useJacStream(walker, nodeId, fields)` hook for consuming server-sent-events from generator-returning walkers and functions. `jac-scale` already auto-wraps any walker/`def:pub` returning a Python generator as `text/event-stream`; these primitives close the loop on the client so apps can render token-by-token output (`yield {"token": "..."}`) or structured agent events (`yield {"type": "tool_call", "detail": "..."}`) as the server emits them. The hook returns `{events, isStreaming, error, start(extra_fields), reset()}`, where `events` is the raw array of yielded objects so callers decide whether to join, filter, or render by type. The `cl{}` block bundler auto-imports the new symbols from `@jac/runtime`, so user code uses them without explicit imports.
