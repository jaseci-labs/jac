- **Breaking: `jac setup desktop` no longer runs sudo or installer commands on the user's behalf**. Previously the command would pipe `curl | sh` to install Rust, run `sudo apt-get/dnf/pacman install` for build tools and webkit/gtk deps, and invoke `xcode-select --install` on macOS -- each gated only by a `[Y/n]` prompt. The command now detects missing dependencies and prints the exact platform-specific install command (Rust via rustup / winget, build tools via `apt-get install build-essential` / `dnf install gcc gcc-c++` / `pacman -S base-devel`, webkit/gtk deps per distro, Xcode CLT via `xcode-select --install`, VS Build Tools via `winget`), then exits. Users run the command themselves and re-run `jac setup desktop`. Rationale: a project-scaffolding command should not escalate privileges or modify system packages. Removed helpers: `_check_and_install_rust`, `_check_and_install_build_tools`, `_check_and_install_linux_dependencies`, `_try_install_linux_build_tools`, `_try_install_linux_system_deps`. New helpers: `_check_rust`, `_check_build_tools`, `_check_linux_dependencies`, `_print_linux_build_tools_instructions`.
