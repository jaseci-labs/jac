- **Feature: Dev-only devtools overlay**: `jac start` now injects a floating devtools overlay on top of the running app. It renders in its own React root inside a Shadow DOM, fully isolated from the host app's component tree and styles, and provides a draggable, tabbed panel shell with an Environment panel (route, mounted module and entry, viewport, and runtime info). The overlay is on by default in dev, can be forced on or off with the `JAC_DEVTOOLS` environment variable, and is omitted from production `jac build` output unless opted in via `[plugins.client.devtools] enabled` or `JAC_DEVTOOLS=1`.
