- **Feature: Dev-only devtools overlay**: `jac start` now injects a floating, draggable devtools overlay on top of the running app, rendered in its own React root inside a Shadow DOM (fully isolated from the host app's tree and styles). It has a Network panel that captures walker and function calls (including cache hits, and, when byllm is used, the LLM calls each request made) and an Environment panel (route, mounted module and entry, viewport, runtime info). The overlay is on by default in dev, can be forced on or off with the `JAC_DEVTOOLS` environment variable, and is omitted from production `jac build` output unless opted in via `[plugins.client.devtools] enabled` or `JAC_DEVTOOLS=1`.
