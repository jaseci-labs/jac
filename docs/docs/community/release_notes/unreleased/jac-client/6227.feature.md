- **Feature: Beta React Native target (`--client react-native`)**: Adds a second mobile path alongside Capacitor: `jac setup react-native`, `jac start --client react-native --dev`, and `jac build --client react-native` drive an Expo/Metro/Hermes scaffold under `.jac/mobile-rn/`. Cross-platform `.cl.jac` source maps familiar HTML tags (`div`, `button`, `input`, …) to React Native primitives at compile time; `.native.cl.jac` files override web-only pieces when present. Includes a shared `JacClientCompiler` pipeline, a native client runtime (`client_runtime_native`), dual-platform examples, and extensive test coverage.
