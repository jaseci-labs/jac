- **Feat: Mobile Target with WebView-based React Native/Expo Support**: Added complete mobile target implementation (`jac build main.jac --client mobile`) that generates React Native apps using Expo Router. The mobile build extends the web target: it first builds the web bundle (TypeScript + React), then scaffolds an Expo project with WebView integration to render the Jac app. Includes `jac setup mobile` for project scaffolding, `[mobile]` configuration section in `jac.toml` for app metadata (bundle ID, display name, platforms, Expo settings), EAS Build integration for production builds (`-b production`), and dev server support (`jac start main.jac --client mobile --dev`) that runs both the web dev server and Expo development client. Templates include proper navigation setup, asset handling, and platform-specific configuration (iOS bundle identifier, Android package name, adaptive icons, splash screens).
