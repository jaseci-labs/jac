- **Add: Pre/Post Invoke Hooks on Model.invoke**: `register_pre_invoke_hook` and `register_post_invoke_hook` allow plugins to intercept every `by llm()` call. Pre-hooks fire synchronously before the LLM call and can raise to block it; post-hooks fire after success. Added `LLMRateLimitExceeded(ByLLMError)` exception for typed rate limit signaling.
