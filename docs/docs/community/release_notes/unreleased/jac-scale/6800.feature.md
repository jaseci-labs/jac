- **Feature: Auto-route client stubs to the right microservice**: In microservice mode the gateway now injects a live `{module: /api/<prefix>}` map (built from the service registry) into the SPA's `__jac_init__` data when it serves `index.html`. Combined with the import-path provenance the compiler stamps on cross-module client stubs, a frontend that just `sv import`s and calls `create_order()` is routed automatically to `/api/orders/function/create_order` - no hand-written `apiCall("orders", "create_order")` helper, no reliance on the gateway's slow fan-out passthrough, and no breakage when two services expose a same-named function. The fan-out passthrough remains as a fallback for builtin routes and older bundles. Monolith apps are unaffected: with no registry the map is empty and stubs use the flat path.
