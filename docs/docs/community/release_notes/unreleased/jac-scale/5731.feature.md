- Optional Loki + Grafana Alloy log aggregation in the Kubernetes monitoring stack: opt in via `[plugins.scale.kubernetes].loki_enabled = true`. Loki single-binary on `emptyDir` storage; Alloy DaemonSet (River syntax) tails `/var/log/pods` and ships to Loki; Grafana gets a **Pod Logs** dashboard. Alloy supersedes Promtail (EOL 2026-03-02).
