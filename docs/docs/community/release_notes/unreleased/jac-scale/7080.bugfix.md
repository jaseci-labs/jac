- **Fix: `--experimental` host build cross-compiles for the pod platform**: `build_release_binary` built the shipped jac binary for the host OS/arch, so deploying from a non-linux host (e.g. macOS arm64) shipped a Mach-O binary into linux pods that could not exec. The host build now passes `-Dtarget` to `zig build` and `BinaryInjector` resolves the pod target (default `x86_64-linux`) when the host is not linux, so the shipped binary always matches the pods. Native builds on a linux host are unchanged.
