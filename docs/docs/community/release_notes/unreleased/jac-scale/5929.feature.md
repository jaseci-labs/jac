- **jac-scale: Replace Motor with PyMongo native `AsyncMongoClient`**: Motor (the third-party async MongoDB driver) reached end-of-life May 2026 and was replaced by PyMongo 4.9+ `AsyncMongoClient`. `MongoBackend.aget` and `acommit` now use a process-level `AsyncMongoClient` singleton (via `_process_cache`) instead of creating a new Motor client per call. This eliminates per-request client allocation overhead and removes the `motor` dependency entirely.
