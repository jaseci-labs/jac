- **Faster hook dispatch under `--scale`**: `JScaleExecutionContext` now seals the runtime hook table on its first initialization, so the Phase 2 single-implementation devirtualization (see the jaclang note for this PR) also applies when serving under `--scale`. `JScaleExecutionContext.init` overrides the core `ExecutionContext.postinit` seam where the seal otherwise fires, so it is triggered here too; the call is idempotent and runs once at the first request. Multi-implementation hooks (e.g. jac-scale's storage backend) stay on the dynamic dispatcher.
