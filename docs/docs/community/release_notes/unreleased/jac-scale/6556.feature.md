- **Feature: Ops Console - Image & Build Freshness (V-06)**: A new **Operations -> Image Freshness** admin page is the truth-teller for image-caching bugs ("did my deploy actually update?"): per app it shows the running image tag(s), the resolved sha256 `imageID` digest(s) across replicas (more than one distinct digest for the same tag = cross-replica DRIFT, highlighted - different nodes pulled different bytes), and the B-01 build identity (`JAC_BUILD_SHA` / `JAC_IMAGE_REF`) injected into pod env, so an operator can confirm a deploy shipped the commit they expect. Reuses the read-only Ops Console `Role` (V-02); pure grouping/drift helpers unit-tested.
