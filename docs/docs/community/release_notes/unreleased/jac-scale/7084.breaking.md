- **Breaking: `jac start --scale --experimental` is removed**: the in-pod/host build is now selected through the `[dev]` jac.toml stanza instead of the CLI flag. To host-build your local checkout and ship it (the old `--experimental` behavior), add `[dev] jaclang_source = "<dir containing jaclang/>"`; for a no-build prebuilt runtime, use a bare `[dev]` (rolling dev binary) or no `[dev]` at all (latest stable release).
