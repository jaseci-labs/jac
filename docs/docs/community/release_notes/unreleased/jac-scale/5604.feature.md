- **Add: Per-User LLM Rate and Budget Limiting**: jac-scale now supports configurable per-user limits on `by llm()` calls via `[plugins.scale.llm_limits]` in `jac.toml`. Supports RPM, RPD, TPM, TPD, daily USD budget, and monthly USD budget. Limits are enforced pre-call (raises on violation), counters are stored in Redis, and usage records are persisted to MongoDB so daily/monthly budgets survive a Redis restart. Requires `jac-scale[data]`.
