- **Fix: ScaleTieredMemory.acommit() now forwards anchor parameter**: `acommit(anchor)` previously ignored the `anchor` argument and always ran a full L1 flush via `commit()` with no arguments. It now passes `anchor` through to `asyncio.to_thread(self.commit, anchor)` so single-anchor commits work as the signature implies. The `_committed` guard is only set to `True` on full commits (`anchor is None`), preventing partial commits from short-circuiting future full commits.
