- **Fix: `@restspec` endpoints crash on unknown query params (#4661)**: Browsers, CDNs, and frontend frameworks routinely append undeclared query parameters (`?v=...` cache busters, tracking params, etc). These now silently bypass walker and function callbacks instead of producing a 500 `TypeError: got an unexpected keyword argument`. `create_walker_callback` and `create_function_callback` precompute accepted field/parameter names from the underlying signature and filter incoming kwargs unless the callable explicitly declares `**kwargs`. Matches FastAPI's default behavior for undeclared query params.
