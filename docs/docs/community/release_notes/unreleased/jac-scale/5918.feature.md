- **Feat: `Jac.find_nodes(ArchType, field_filter)` builtin**: Walker-level API to query persisted nodes by type and optional field predicates without any storage-backend awareness. On MongoDB (jac-scale) the call delegates to `MongoBackend.query_by_type` so only matching documents are fetched (IXSCAN). On SQLite or in-memory backends it falls back to a Python-side type and field filter over the in-memory anchor dict. Results are loaded into the L1 cache so edge operations work immediately. Supports MongoDB-style operators (`$lt`, `$lte`, `$gt`, `$gte`, `$ne`, `$eq`) in the Python fallback path. Depends on PR #5917.
