- **Plugin IPC: OS capabilities for webview apps**: Desktop apps expose a plugin host with seven built-in plugins (`path`, `window`, `fs`, `notification`, `clipboard`, `dialog`, `shell`) to JS via `window.__jac.invoke()` and `window.__jac.on()`, with per-plugin security gates (allow-lists, opt-in toggles, and deny-all shell defaults) configured under `[plugins.desktop.plugins]` in `jac.toml`.
- **Plugin SDK: typed client API and registry bootstrap**: Adds a plugin manifest, registry, and typed `@jac/desktop` client SDK; the native host bootstraps built-in plugins from the registry at startup and transpiles plugin `.jac` sources at build time.
