- **Feature: CEF desktop target (`desktop-cef`)**: New `CefDesktopTarget` provides a Chromium Embedded Framework renderer alongside the OS-native webview. Set `engine = "cef"` under `[plugins.desktop]` and build with `jac build --client desktop-cef`. The loopback server and OAuth broker are identical to the native target; CEF integration is Jac-native (`cef.na.jac` + `libcef_dispatch.so` + `cef-subprocess`) with a pinned CEF 119.x binary fetch. (jaseci-labs/jaseci#6572)
