- **Feature: CEF desktop target (`cef`)**: New `CefDesktopTarget` provides a Chromium Embedded Framework renderer alongside the OS-native webview. Set `engine = "cef"` under `[plugins.desktop]` and build with `jac build --client cef`. The loopback server and OAuth broker are identical to the native target; CEF integration is Jac-native (`cef.na.jac` + `libcef_dispatch.so` + `cef-subprocess`) with a pinned CEF 119.x binary fetch. The fetch now pins the *exact* CEF build and verifies the download against an in-tree digest (`cef_sums.lock`) instead of trusting the live CDN index, and the dispatch layer asserts the loaded `libcef.so` matches the verified ABI version at startup - a mismatch aborts with a clear message rather than corrupting memory inside `cef_initialize`. (jaseci-labs/jaseci#6572)
