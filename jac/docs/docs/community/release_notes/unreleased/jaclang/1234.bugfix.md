# Release Note

- Fixed three bugs in the C-to-Jac transpiler (c2jac):
  - Added missing `_struct_classdef` declaration in `CMapper` interface that caused `AttributeError` when transpiling structs
  - Fixed chained assignments (e.g., `*a = *b = *c = 0`) which previously caused Internal Compiler Error
  - Fixed nested struct type resolution so fields like `Point origin` are correctly typed instead of defaulting to `int`
