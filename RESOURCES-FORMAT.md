# RESOURCES-FORMAT.md

`RESOURCES.md` is a curated list of high-quality, high-trust resources used to
ground the teaching. Never trust parametric knowledge — cite real sources.

## Format

```markdown
# Resources

## <Category name>

- **<Title>** — <one-line description>. <URL>
  - <optional: why it's trusted / what it's best for>
```

Group by category. Mark the single best resource per topic with **(primary)**.
Prefer official docs, canonical references (Rustonomicon, RFCs), and the project's
own implementation docs over blog posts.
